This is a TypeScript open-source game which is a port of the ActionScript version, following the same concepts, graphics, and architecture.

+ **Category:** Puzzle.
+ **Platform:** Web.
+ **Language:** TypeScript.
+ **Technologies:** PixiJs, RobotlegsJs, RobotlegsJs-Pixi, Palidor.
